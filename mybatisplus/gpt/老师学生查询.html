我现有2张表，1张教师表，1张学生表，这两张表有关联关系，请帮我使用mybatis-plus中写出2张表的关联查询，请使用mapper.xml的方式来写2张表的关联查询，请给出完整的代码。

假设教师表为 teacher，包含字段 id、name 和 age，学生表为 student，包含字段 id、name、age 和 teacher_id，其中 teacher_id 为学生表关联教师表的外键。

下面是使用 MyBatis-Plus 的 XML 配置文件实现两张表的关联查询的示例代码：

  1. 创建 VO 类 TeacherStudentVO，包含教师信息和对应的学生列表：
public class TeacherStudentVO {
    private Long id;
    private String name;
    private Integer age;
    private List<Student> students;
    // getter, setter and constructor
}

java
复制成功
  1. 在 teacher 的 mapper 文件中编写关联查询的 SQL 语句：
<!-- 查询所有教师及其对应的学生列表 -->
<select id="findAllTeachersWithStudents" resultMap="teacherStudentVOResultMap">
    SELECT
        t.*,
        s.id AS student_id,
        s.name AS student_name,
        s.age AS student_age
    FROM
        teacher t
        LEFT JOIN student s ON t.id = s.teacher_id
    ORDER BY
        t.id ASC,
        s.id ASC
</select>

<!-- 配置 resultMap，将结果集映射到 TeacherStudentVO 对象中 -->
<resultMap id="teacherStudentVOResultMap" type="com.example.TeacherStudentVO">
    <id column="id" property="id" />
    <result column="name" property="name" />
    <result column="age" property="age" />
    <collection property="students" ofType="com.example.Student">
        <id column="student_id" property="id" />
        <result column="student_name" property="name" />
        <result column="student_age" property="age" />
    </collection>
</resultMap>

xml
复制成功
  1. 在 TeacherMapper 接口中定义关联查询的方法：
public interface TeacherMapper extends BaseMapper<Teacher> {
    List<TeacherStudentVO> findAllTeachersWithStudents();
}

java
复制成功
  1. 调用 mapper 方法获取查询结果：
List<TeacherStudentVO> teacherStudentVOList = teacherMapper.findAllTeachersWithStudents();
